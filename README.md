Telecom Customer Churn Analysis & Retention Dashboard
📌 Project Overview

This project focuses on analyzing customer churn behavior in the telecom industry using Python, SQL, and Power BI.

The main objective of this project is to identify customer churn patterns, revenue risk factors, and retention opportunities through data cleaning, SQL analysis, and interactive dashboard development.

The project follows a complete end-to-end data analytics workflow including:

Data Cleaning using Python
Data Analysis using SQL
Data Visualization using Power BI
Business Insights & Retention Analysis

📂 Project Workflow
1️⃣ Data Cleaning (Python)

The raw telecom dataset contained:

Missing values
Inconsistent categorical values
Unstructured service-related columns
Duplicate and formatting issues
Cleaning Steps Performed:
Handled missing values
Fixed data types
Standardized categorical values
Removed inconsistencies
Prepared data for SQL and Power BI analysis
Result:

The cleaned dataset became more structured, analysis-ready, and suitable for business intelligence reporting.

2️⃣ SQL Analysis

SQL queries were used to analyze:

Overall churn rate
Contract-wise churn behavior
Revenue loss patterns
Tenure-based customer retention
Payment method impact on churn
High-risk customer segments
Key SQL Insights:
Month-to-month contract customers showed the highest churn behavior.
Customers with low tenure had a higher probability of leaving.
Electronic check users were more likely to churn.
High monthly charges were strongly associated with churn risk.

📊 Business Insights
📌 Contract Analysis

Customers with month-to-month contracts showed significantly higher churn rates compared to long-term contract customers.

Business Impact:

Long-term contracts improve customer retention and reduce churn risk.

📌 Revenue Risk Analysis

Customers with higher monthly charges demonstrated higher churn behavior.

Business Impact:

High-value customers leaving the service can directly impact company revenue.

📌 Customer Tenure Analysis

New customers and low-tenure customers were identified as the most vulnerable segment.

Business Impact:

The company should focus on improving onboarding and early customer engagement.

📌 Payment Method Analysis

Electronic check payment users showed the highest churn percentage.

Business Impact:

Payment experience and billing methods may influence customer satisfaction.

📌 Service Dependency Analysis

Customers without support and security-related services were more likely to churn.

Business Impact:

Additional support services can improve customer retention and engagement.
